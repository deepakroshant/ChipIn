Complete Navigation Map — Chip In

Every screen, thoughtfully placed. Minimal taps to do anything.

🏠 Home Tab

Dashboard — first thing you see
• Net balance card — big, clear: "You're owed $240" or "You owe $80" in warm amber/red
• Quick-settle list — top 3 friends you owe/are owed, one-tap to Interac
• Activity feed — live stream of all group/friend expenses with timestamps
• Floating + button — always visible, opens Add Expense sheet

👥 Groups Tab

Groups list → Group detail
• Each group shows: custom emoji/colour, name, your balance, last activity
• Group detail: expense list, member balances, settle-up summary
• Expense detail: full split breakdown, comments, edit/delete
• Friends view (inside Groups): 1-on-1 balances outside of groups

➕ Add Expense (floating button — everywhere)

Fast, 3-tap expense entry
• Amount → big numpad, feels like Apple Pay
• Who paid + split with → pick group or friends
• Split method: Equal / % / Exact / By item / By shares
• Category → emoji-tagged (Food, Travel, Rent, Fun, Utilities, Other)
• Receipt scan → tap camera icon, AI fills the form
• Recurring toggle → set frequency (weekly/monthly)
• Note → optional description

📊 Insights Tab

Beautiful spending analytics
• Monthly overview — total spent, owed, settled this month
• Category breakdown — animated donut chart with warm colours
• Spending trends — 6-month bar chart
• Top spender in each group — fun social stat
• Settle history — log of all past settlements

👤 Profile Tab

Personalization + Settings
• Avatar + name — custom profile photo or Memoji
• Appearance — Dark / Light / System, accent colour picker (5 warm palettes)
• Default split — set your preferred split method per group
• Notifications — smart nudge timing, settle reminders on/off
• Currency — CAD default, can add travel currencies
• Widgets — configure Home Screen + Lock Screen widgets
• App icon — 4 alternate icons (dark, light, minimal, colourful)
• Privacy — biometric lock, hide balances mode

💸 Settle Up Flow

One tap to pay
• Tap "Settle" on any balance → shows exact amount
• Interac e-Transfer — deep link opens banking app with amount pre-noted
• Mark as settled manually — if paid cash or outside app
• Confirmation screen with confetti 🎉
Does this navigation feel complete and minimal? Everything accessible in 2–3 taps max. No buried menus.
