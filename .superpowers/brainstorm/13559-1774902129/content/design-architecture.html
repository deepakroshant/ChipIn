Design Section 1: Architecture & Navigation

How the app is structured at the highest level

App Structure

Frontend

SwiftUI + SwiftData (local cache + offline). Targets iOS 17+. Modular feature folders.

Backend

Supabase — PostgreSQL database, real-time subscriptions, Auth (magic link + Apple Sign-in), Edge Functions for receipt AI.

Ads

Google AdMob — banner ads on list screens, interstitials between major actions (max 1/session). No ads inside expense creation flow.

Receipt AI

iOS Vision framework (free, on-device OCR) → parsed via Supabase Edge Function with OpenAI. No data leaves device unnecessarily.
